깃 초기 설정
깃을 저장할 파일 경로에 git bash로 저장한다. 
-> ex) 
	cd d: 
	cd dev
	ls -al : 이 커멘드는 해당 폴더에 있는 directory 리스트를 보여준다


그후, 
1. git init
- 저장할 파일의 최종 경로에 init을 한다. 	:	init된 공간이 시작점이 됨	
2. git add .			  	:	init된 dir의 파일을 저장하기 위해서 체크하는 느낌
3. git commit -m "first commit"	  	:	코멘드 남기기
4. git branch -M master		  	:	master *확인 하기*
5. git remote add origin 주소(git repository)
6. git push -u origin master


깃 처음으로 repository에서 PC로 연결하기
git clone 주소


깃 초기 설정 이후
